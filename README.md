## Usage

## Fast
- Clone the repo
- Run yagna market
- Run <code> python3 run_ripper.py </code>
### Standard
- Clone repo
- <code> docker build . -f jtr.Dockerfile </code>
- <code> gvmkit-build <image id> </code>
- <code> gvmkit-build <image id> --push </code>
